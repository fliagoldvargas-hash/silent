# Silent Protocol Social Image Template

Use this template for every X/Twitter post image.

## Fixed Visual Rules

- Format: horizontal landscape image, wider than tall.
- Background: use `public/social/silent-x-template-reference.png` as the fixed visual reference/base.
- Background look: deep black with subtle dotted/noise texture and faint smoky neon-pink abstract lines.
- Palette: black, hot pink glow from the Silent Protocol logo, off-white title, muted gray subtitle.
- Logo: Silent Protocol pink lightning logo at the top center.
- Layout:
  1. Logo centered at the top.
  2. Title centered below the logo.
  3. Subtitle centered below the title.
- Mood: premium, dark, technical, quiet, private, high-signal.
- Do not add people, coins, charts, extra icons, extra logos, watermarks, random objects, or additional text.
- Do not change composition, background style, logo placement, palette, or typography direction.
- The production generator is `scripts/generate-social-posts.ps1`.

## Only Editable Fields

- Title
- Subtitle
